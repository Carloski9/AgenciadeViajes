# ✈️ Agencia de Viajes Delmiro

Landing page desarrollada como práctica de Lenguaje de Marcas. Es una web interactiva y totalmente responsive sobre una agencia de viajes ficticia.

## 🛠️ Tecnologías utilizadas
* HTML5 Semántico
* CSS3 (Flexbox y CSS Grid)
* Animaciones (Animate.css y GSAP)

## 🎯 Características
* Diseño adaptable a dispositivos móviles.
* Uso de variables CSS para el esquema de colores.
* Interfaz moderna con gradientes y animaciones suaves.
